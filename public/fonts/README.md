# Retronovel Fonts

## ⚠️ IMPORTANT: Add dogica.ttf here!

Place the **dogica.ttf** file in this directory:
`public/fonts/dogica.ttf`

## Why Dogica?

Dogica is a pixel-perfect bitmap font specifically designed for retro games and pixel art. It will make your text crisp and readable at small sizes!

## Where to get it

Download from: **https://www.dafont.com/dogica.font**

Or search "dogica font" on Google.

## After adding the font

1. Place `dogica.ttf` in `public/fonts/`
2. Restart the dev server (`npm run dev`)
3. Text should now be pixel-perfect!

## Custom Fonts

Users can also upload their own fonts through the Settings tab in the editor.
Any pixel art / bitmap font will work great!

## Recommended Pixel Fonts

- **Dogica** (recommended!)
- m5x7
- m3x6  
- Pixel Operator
- Press Start 2P
- VT323

All of these are designed for pixel art and will render crisp at small sizes.
