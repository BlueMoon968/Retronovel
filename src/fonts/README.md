# Fonts Directory

## Default Font

Place **dogica.ttf** in this directory for pixel-perfect text rendering.

Dogica is a bitmap font designed for pixel art and retro games.

## File Structure

```
src/fonts/
├── dogica.ttf          (Required - place your font file here)
└── README.md           (This file)
```

## How to Get Dogica

Download from: https://www.dafont.com/dogica.font

Or use any pixel art font you prefer!

## Custom Fonts

Users can upload custom fonts through the Settings tab in the editor.
The default font (dogica.ttf) is always available as fallback.
